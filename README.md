# Game discovery APP

Discover all interesting games here

### FEATURES

With Game discovery APP you can:

- Filter and search for games
- view games

### Requirements

- Node v12.0.0
- React.js framework
- Typescript

### Installation

1. Clone the repository

2. Install dependencies
   `npm install`

3. Run app
   `npm start`

4. To build the app
   `npm run build`

5. Testing

`npm run test`
